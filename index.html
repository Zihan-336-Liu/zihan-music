使用说明

  1. 将您的音乐文件上传到GitHub仓库
  2. 在上方输入框中填写GitHub用户名、仓库名称等信息
  3. 点击"加载播放列表"按钮
  4. 播放列表将自动生成，点击歌曲即可播放
  5. 使用控制按钮控制播放：播放/暂停、上一首、下一首等

GitHub + jsDelivr URL格式:

https://cdn.jsdelivr.net/gh/用户名/仓库名@分支/文件路径

例如: https://cdn.jsdelivr.net/gh/johndoe/music@main/song1.mp3

部署到GitHub Pages:

  1. 在GitHub上创建新仓库
  2. 将本HTML文件保存为index.html并上传到仓库
  3. 上传您的MP3文件到仓库
  4. 转到仓库Settings → Pages → 选择main分支 → Save
  5. 等待几分钟后，您的播放器就可以通过https://用户名.github.io/仓库名 访问了
